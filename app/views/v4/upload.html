Upload a file

File types accepted are Word, PDF, JPG, BMP, PNG and TIF.

The combined size of all your files must be less than 10 MB.

Help if your files are too big

If the combined size of the files you upload is bigger than 10MB you’ll see a message saying they cannot be sent.

If a single file is bigger than 10MB, reduce the file size if you can and try again.

If the combined size of more than one file is bigger than 10MB, try sending fewer files. Use the service again to send any remaining files.

If you still cannot upload and send your files you can post them instead. Call 0800 121 4433 (Textphone: 0800 121 4433) if you do not know the freepost address to send them to.
